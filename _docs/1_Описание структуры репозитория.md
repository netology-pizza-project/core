Описание структуры репозитория
------------------------------

### Структура проекта

```.
├── README.md 
├── _dev_diaries -- дневники разработки членов команды
│   └── ...
├── _docs        -- текущая документация
│   └── ...
├── backend      -- компоненты backend
│   └── ...
├── frontend     -- компоненты frontend
│   └── ...
```

### Правила внесения изменения в файлы проекта

#### Общие правила
1. Ветка `main` -- основная ветка разработки проекта. В ней содержится текущая стабильная версия проекта, а также
изменения, внесённые по результатам спринта разработки.
2. Ветки `sprint-{порядковый номер спринта по Jira}` -- ветки с изменениями, вносимыми по задачам в рамках 
соответствующего спринта. Ветка отводится в начале спринта от ветки `main`, затем члены команды отводят от неё
неё собственные ветки разработки по задачам. Вливание в `main` осуществляется без удаления.
3. Ветки `PIZ-{номер задачи из Jira}-{краткое описание изменений}` -- ветки с изменениями, вносимыми в рамках
отдельной задачи спринта. В случае, если изменения по задаче незначительные, разрешается вливать в ветку спринта без
ревью. Если качество изменений значительное -- требуется Merge Request с код-ревью: для фронтенд -- Владом Вихровым, 
для бэкенд -- Алексеем Ермаковым. Вливание ветки осуществляется с удалением.
4. Ветки `dev-diary-{имя члена команды}-sprint-{порядковый номер спринта по Jira}` -- ветки с дневниками разработки
членов команды. Отводится от `main` и вольным образом заполняется в течение спринта. В конце спринта ветка без
удаления вливается обратно в `main` без удаления.

#### Примерная схема дерева версий проекта

```
Разработка:
main ────────────────────────────────────────────────────────────────────────────────────────────────────────
   └── sprint-2 ─────────────────────────────────────────────────────── (конец спринта)^   └── sprint-3
              └──PIZ-1-add-new-endpoint ─ (выполнение задачи 1)^   |                                  └── ...
              └──PIZ-2-add-another-endpoint ─ (выполнение задачи 2)^

Ведение дневников
main ───────────────────────────────────────────────────────────────────────────────────────────────
   └── dev-diary-ermakov-sprint-2 (конец спринта)^  └── dev-diary-ermakov-sprint-3 (конец спринта)^ 
   └── dev-diary-vikhrov-sprint-2 (конец спринта)^  └── dev-diary-vikhrov-sprint-3 (конец спринта)^
```
